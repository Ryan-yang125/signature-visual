# Motion Direction

Motion is authored change through time. Give it phases, hierarchy, and recovery so the visual communicates more than perpetual activity.

## Write a motion score

Use five phases for a new direction:

| Phase | Purpose | Typical energy |
| --- | --- | --- |
| Rest | establish a complete composition | 10–25% |
| Wake | acknowledge entry or intent | 25–45% |
| Develop | reveal the governing rule | 45–70% |
| Peak | show one decisive transformation | 70–100% |
| Recover | return with residue or new knowledge | 15–35% |

Name what visibly changes in every phase: position, density, topology, material, camera, light, annotation, or history. A phase with no meaningful visual change can be removed.

## Select a temporal character

- **Breathing:** asymmetric expansion and contraction with small phase drift.
- **Accreting:** marks accumulate until a threshold changes the structure.
- **Relaying:** activity passes through nodes with causal timing.
- **Eroding:** boundaries lose coherence and later re-form.
- **Folding:** a surface or topology reveals a hidden relation.
- **Scanning:** a measured window exposes layers or values.
- **Swarming:** many agents align, split, and rejoin around a rule.
- **Stratifying:** time becomes persistent layers rather than a loop.

Tie the character to the core verb pair. Use one primary character and a quieter secondary rhythm.

## Timing hierarchy

Define three bands:

```text
MACRO   full arc or loop; the visitor can perceive its structure
MESO    object, cluster, or layer changes that support each phase
MICRO   flicker, grain, ripple, or jitter that gives material life
```

Keep micro motion below the attention threshold when copy is primary. Synchronize selected events across bands so the peak reads clearly.

## Curves as behavior

Describe easing through the phenomenon before choosing an equation:

- elastic return for tensioned membranes;
- viscous settling for fluid or particulate systems;
- stepped acquisition for instruments;
- ballistic departure and drag for emitted marks;
- critical damping for pointer recovery;
- delayed neighbor response for growth and topology.

Avoid applying the same smooth ease to every property. Position, opacity, deformation, and camera often need different response times.

## Interaction score

Map input into the existing arc:

```text
REST        system is complete without input
APPROACH    proximity reveals affordance
ENGAGED     input amplifies or redirects the core rule
RELEASED    energy decays through the material model
RECOVERED   the signature composition returns, with optional residue
```

Pointer movement should reveal behavior, not merely make objects chase the cursor. Cap response magnitude and velocity. Preserve a stable exit path even when the pointer leaves quickly.

## Scroll and timeline direction

For scroll-linked work:

- assign one narrative change to each segment;
- pin only when spatial continuity makes the relationship clearer;
- keep progress deterministic so any percentage produces a valid frame;
- avoid long spans where only speed or opacity changes;
- provide a coherent entry and exit frame for back-navigation.

## Reduced motion

Preserve the idea through one of these strategies:

- select the strongest authored state;
- replace travel with immediate state changes;
- reduce amplitude and remove camera movement while retaining material life;
- convert the arc into a small set of user-triggered steps;
- show accumulated history as a static composition.

Reduced motion is an alternate direction for time, with the same signature rule and information hierarchy.
