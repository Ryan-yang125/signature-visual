# Direction Case Study: Local Evidence, Regional Confidence

This worked example shows the V3 section workflow before implementation. The brief asks for one concise evidence visual on a climate-intelligence page with warm paper, dark green type, and restrained red annotation.

## Surface and scope

```text
TIER            section
PAGE JOB        explain how local observations support a regional conclusion
CONTENT AXIS    every conclusion remains traceable to its evidence
TARGET EVIDENCE observations share methods, disagree, and gain authority together
BRAND SIGNAL    warm paper, botanical green, editorial correction
BOUNDARIES      inline section, real labels, 390px mobile, reduced motion
OUTPUT HISTORY  nearest prior output: horizontal paper strata with a probe
```

Pattern Language and renderer references remain closed during search.

## First evidence-led direction

### Evidence Delta

```text
GOVERNING RULE  local marks settle into layers; every conclusion retains its strata
SILHOUETTE      horizontal cross-section under a large quiet paper field
MATERIAL        dry green ink, translucent sediment, rare red conflict mark
TIME            accumulation with persistent residue
INTERACTION     probe one layer to reveal contributing observations
TYPE ROLE       annotation around the section
```

The direction fits the target and brand. Output-distance review finds a structural collision with the nearest prior visual:

| Axis | Evidence Delta vs nearest output |

| Silhouette | same horizontal strata and quiet upper field |
| Spatial grammar | same accumulation into one lower boundary |

Five high-salience axes converge. The section workflow now earns one structural branch.

## Distance-triggered branch

### Witness Weave

```text
GOVERNING RULE  a regional claim gains width only where independent observations cross one shared audit window
SILHOUETTE      three narrow source bands cross a square void and open into one broad conclusion
MATERIAL        dry green threads, translucent audit glass, one red correction knot
TIME            held inspection; no ambient loop
INTERACTION     selecting a source opens its exact path through the weave
TYPE ROLE       observation labels sit inside the contributing bands
```

The branch changes artifact stance, spatial grammar, temporal model, interaction meaning, and type role. Warm paper and restrained editorial color remain as brand continuity.

## Freeze decision

Witness Weave is selected because the crossing rule makes independent evidence and shared method visible in one held frame. Its strongest counterevidence is the familiarity of braided-path diagrams. The implementation must keep the square audit window, unequal source widths, and label-bearing bands perceptible at thumbnail scale.

Signature rule: **every regional claim visibly contains the independent paths that crossed its audit window.**

Pattern Language opens after this freeze. “Trace causality” helps test route hierarchy, and “calibrate a hidden state” tests whether the audit window measures real confidence. Both act as critique operators; the direction keeps its own geometry and name.

## Authored system

### Composition

- Three asymmetrical source bands enter from different edges.
- A square audit void sits off-center and creates the primary negative space.
- The conclusion band exits wider while preserving visible internal paths.
- Mobile rotates the audit window and stacks source labels on a top rail; it does not scale down the desktop topology.

### Material

- Ground: warm uncoated paper with screen-fixed fine grain.
- Body: dark-green dry threads whose width represents evidence coverage.
- Counterpoint: clear audit glass with a hard graphite edge.
- Exception: red appears only where two sources disagree on method or value.
- History: selected paths stay legible until another source is selected; no automatic accumulation occurs.

### Temporal and interaction structure

The default is an authored still. Pointer, touch, or keyboard selection enters an inspection state: one source path widens, unrelated paths soften, the audit window exposes its method, and the conclusion shows that source's exact contribution. Cancellation returns to the held overview. The primary section action can commit the selected source as a persistent comparison state when the product surface provides that operation.

### Renderer route

SVG leads because labeled paths, topology, focus order, and deterministic inspection carry the direction. CSS supplies paper texture. A raster or GPU layer would add fill cost without advancing the governing rule.

## Semantic capture protocol

- desktop held overview;
- desktop source inspection;
- desktop conflicting-source state;
- desktop cancellation aftermath;
- mobile held overview and source inspection;
- reduced-motion overview and keyboard-selected source;
- empty, typical, and malformed data states.

## Critique and revision

First derived sheet observation: the full-size inspection state is clear, while its 160px thumbnail loses the selected source inside the conclusion band.

Cause: selection changes opacity and label detail while preserving the same band width.

Revision: the selected source opens a narrow white channel through the conclusion band and shifts the audit-window crop by eight percent. The new thumbnail and silhouette show the inspection state immediately, while the held frame and mobile label rail remain stable.

Counterevidence: the red disagreement knot still competes with the selected-source channel in the malformed-data state. The next check isolates that state and tests a graphite correction mark before changing the brand signal color.
