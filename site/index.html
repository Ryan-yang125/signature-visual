Adversarial selection

The winner must survive hostile questions.

  1. Remove the paletteDoes the silhouette still belong to the product?
  2. Freeze the motionDoes the resting frame hold hierarchy and tension?
  3. Swap the rendererDoes the thesis survive outside one familiar effect?
  4. Forecast convergenceWhat artifact, material, time model, and interaction will another run choose first?
  5. Break the causal clusterDoes the winner cross at least two causal axes from that forecast?
  6. Invite an external directionCan a new candidate beat the current choice under the same rubric?

Use patterns as critique operators.

Each operator pressures one part of the chosen system. It opens a controlled mutation without replacing the visual thesis.

Coherence operator

Can local behavior reveal one larger rule?

Apply when the frame contains many actors with no readable collective consequence.

Pressure: force law / corridor / density / edge behavior
Boundary operator

Where does identity become permeable?

Apply when an object has presence but its response carries little meaning.

Pressure: silhouette / porosity / interior / recovery
Instrument operator

What becomes measurable through interaction?

Apply when invisible state needs a precise, bounded, and recoverable response.

Pressure: scale / sampling / control / signal decay
TRACETRACETRACETRACE
Memory operator

Which past action must remain visible?

Apply when recovery erases evidence that the product should preserve.

Pressure: strata / annotation / erosion / reveal path
Deposition operator

What material trace should computation leave?

Apply when motion feels weightless and needs accumulation, drag, or residue.

Pressure: solvent / bleed / granulation / drying sequence
Threshold operator

Where does change become undeniable?

Apply when the motion has activity but no decisive event or afterimage.

Pressure: contact / interference / threshold / falloff
