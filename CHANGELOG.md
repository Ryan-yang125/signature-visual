# Changelog

## 2.0.0 — Visual Director

- Rebuilt the core workflow around a visual thesis, three divergent Direction Cards, evidence-based selection, and an authored composition/material/motion/interaction specification.
- Added ten technology-independent pattern cards with mutation axes, capture checkpoints, cheap-look signals, and source-distance rules.
- Added dedicated visual direction, composition, material, motion, interaction, reference-study, failure-diagnosis, renderer, integration, and Visual QA references.
- Replaced finished-effect starters with neutral Canvas, Three.js, WebGL, and SVG runtime shells that expose deterministic controls and full lifecycle cleanup.
- Added a deterministic Playwright QA CLI for timeline, ambient, pointer, data, mobile, and reduced-motion captures, plus contact sheets, error diagnostics, manifests, and image hashes.
- Added independent evaluation scenarios and a forward-test rubric.
- Redesigned the website and live cases to demonstrate divergent visual identities, five-state motion scores, pattern variation, and critique-driven revision.
- Added production manifests for each desktop score, mobile peak, and reduced-motion still, with console, page, and network failures enforced in CI.
- Self-hosted the showcase fonts and Three.js runtime, added GPU fallback verification, and excluded test harnesses from production builds.
- Added named-seed normalization, one cross-renderer pointer contract, semantic SVG controls, keyboard behavior, and mobile target-size/contrast checks.
- Added universal Skill metadata, CI, research acknowledgements, and a worked direction case study.

## 1.0.0 — Initial Release

- Added renderer routing, four computational visual families, lifecycle-aware starters, four live examples, and a static showcase site.
